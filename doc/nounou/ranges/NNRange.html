NNRange

Related Docs: object NNRange | package ranges

class NNRange extends NNRangeSpecifier with LoggingExt

NNRange is the central class for specifying frame-based data sample timepoints (trait NNRangeSpecifier). Various constructor methods for this and other NNRangeSpecifiers are located within the convenience class nounou.NN.

See also

NNRangeTs range defined in timestamp units

NNRangeAll marker for total sample range

NNRangeValid marked to be a valid range (not outside of defined data)

NNRangeInstantiated marked to be a real range (not default, etc.)
